# Offer Teardown & Value Ladder

> Reconstruct a business's ladder of offers — from free bait to flagship — and
> judge how irresistible the core offer really is.

## What it does

Maps the full value ladder (lead magnet → tripwire/entry → core offer →
upsell/continuity) and evaluates the strength of the core offer using
offer-design principles (clear promise, value stacking, pricing/anchoring, risk
reversal, urgency/scarcity, bonuses). Especially useful for info-products,
courses, coaching, communities, and DTC bundles.

## When to use this skill

- "Break down their offer / value ladder."
- "How is this course/coaching/info-product offer structured?"
- "Is this offer strong? What's missing?"
- "Map the ascension path from free to flagship."

## Inputs needed

- **Required:** the site or sales/landing page URL.
- **Helpful:** the niche, price points if not public, and whether the user wants
  a critique, a comparison, or ideas to improve their own offer.
- If the page is a single sales page, map what's visible and note ladder steps
  that are likely but not shown (e.g. backend upsells).

## Method

1. **Find the entry points** — lead magnet / freebie / free webinar / free trial.
   What's the bait and what does it cost the visitor (email, time)?
2. **Identify the tripwire / entry offer** — any low-ticket first purchase that
   converts a lead into a buyer.
3. **Pin down the core offer** — the flagship product/program: what's promised,
   what's included (modules, calls, community, templates), the format, and price.
4. **Trace ascension** — upsells, order bumps, higher tiers, mastermind,
   done-for-you, continuity/membership. Map the ladder rungs and price jumps.
5. **Grade the core offer** on the levers of an irresistible offer:
   - **Promise / dream outcome** — specific and desirable?
   - **Perceived likelihood of achievement** — proof, mechanism, guarantees.
   - **Time to result** — fast or vague?
   - **Effort & sacrifice** — does it lower it (done-for-you, templates)?
   - **Value stack** — components and how value is communicated vs price.
   - **Price & anchoring** — anchor, payment plans, "today only" framing.
   - **Risk reversal** — guarantee strength.
   - **Urgency / scarcity** — real, manufactured, or absent.
6. **Diagnose** — the ladder's gaps (missing rung, no tripwire, weak guarantee,
   no continuity) and the core offer's weakest lever.

## Quality bar

- Quote the actual promise and guarantee wording before grading them.
- Distinguish offers you can see from rungs you're inferring (backend upsells).
- Flag manipulative/fake-scarcity tactics honestly rather than praising them.
- No invented prices — mark "not public" where the page hides them.

## Common mistakes to avoid

- Listing features instead of evaluating the offer's persuasive structure.
- Missing the entry/tripwire rung that turns leads into first-time buyers.
- Praising urgency that's obviously fake (perma-"ends tonight" timers).
